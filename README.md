# Smart University
The tools management system built with React, Vite, and Node.js.

## Features
- Dashboard for monitoring university tools and resources.
- Management of inventory, stock, and transfers.
- User management and authentication.
- Asset tracking and reports.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/elie-iradukunda/smart_university.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend Setup
1. Navigate to the `backend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
